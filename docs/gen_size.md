## 一、详情页规范（应用商店详情页）

各渠道在应用商店详情页需要提供：截图（5图）、视频+封面、其他更新图（ICON、推广图等）

### 外渠（第三方平台）

#### TapTap

| 素材类型 | 规格要求 |
|----------|---------|
| **截图（5图）** | 横版：16:9，≥1280×720 px<br>竖版：9:16，≥720×1280 px |
| **视频** | 分辨率 ≥1280×720，推荐 1920×1080<br>格式：.mp4，编码 H.264 / HEVC（H.265）<br>大小 ≤500MB<br>横版比例 16:9，分辨率 ≥1280×720<br>竖版比例 9:16，分辨率 ≥720×1280 |
| **视频封面** | 900×600 px，JPG/PNG，无文案 |
| **ICON** | 512×512 px，PNG |
| **推广图 / 社区头图** | 1920×1080 px，仅 logo，无文案 |
| **带文案封面图** | 900×600 px |

---

#### 4399

| 素材类型 | 规格要求 |
|----------|---------|
| **截图（5图）** | 960×1600 或 1600×960 |
| **视频** | 分辨率最低 720×405，建议 1280×720<br>格式：MP4，大小 ≤1G<br>时长 ≥30秒，建议 40秒~1分30秒<br>内容须展现真实游戏画面（不可纯CG） |
| **视频封面** | 720×405 px（或新版 1280×720） |
| **ICON** | 512×512 px，PNG |
| **推广图** | 1950×500 px，层次分明（带背景、角色）×1张 |
