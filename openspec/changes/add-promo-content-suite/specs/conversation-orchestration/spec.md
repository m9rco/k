## MODIFIED Requirements

### Requirement: 意图识别与白名单分发
系统 SHALL 仅就预设意图（换背景/换角色/换文案/**平台适配（切尺寸/适配尺寸）**/**生成 icon**/生视频/**联网搜索/图片搜索**/下载打包/**宣发文案生成**/**文字叠加**/**批量变体生成**/**投放前合规审查**）调用对应工具，其余请求礼貌拒绝并列出能力清单；其中「物料爬取」已移除（由图片搜索替代）。「平台适配」意图 SHALL 默认分发到平台适配工具（图生图重绘 + 比例一致时裁剪快路径的智能路由），而非直接调用纯裁剪工具；纯裁剪工具继续保留供前端手动兜底与比例一致快路径复用。工具调用的完成事件 SHALL 结构化携带其所产生长任务的标识，使前端能够即时定位并订阅该任务的进度，而不必等待本轮对话结束。系统 SHALL 减少 tools=0 出现频率：当意图明确且关键参数充足时必须直接调工具，非关键参数可合理推断而无需澄清。为降低弱会话模型的理解偏差，意图识别 SHALL 采用**双层结构**：在调用会话模型前先做一层**服务端确定性预分类**（仅提示、不强制路由），将预判作为结构化「意图提示」注入本轮上下文以强引导模型选对工具；会话模型 SHALL 仍是最终决策者，可推翻该提示。

新增的四类意图 SHALL 按如下归类分发：「写文案 / 出广告语 / 想几条卖点 / 投放短文案」→ `generate_copy`；「加个 CTA / 打个折扣角标 / 加定档大字 / 贴 LOGO」→ `overlay_text`；「多出几版 / 出 N 个不同风格 / 批量变体测一测」→ `generate_variants`；「这能不能投 / 帮我审一下合不合规 / 有没有敏感词」→ `review_compliance`。确定性预分类 SHALL 为这四类意图提供关键词/句式提示，但最终仍由会话模型决策。

#### Scenario: 命中白名单意图
- **WHEN** 用户请求命中预设意图且关键参数充足
- **THEN** 系统在本轮直接调用对应工具，不先发文字确认再调用

#### Scenario: 意图充足时不过度澄清
- **WHEN** 用户说"帮我换个背景"且工作区只有一张图
- **THEN** 系统直接推断使用该图作为 source，调用 edit_image，而非先问"请问要换哪张图"

#### Scenario: 搜索意图纳入白名单
- **WHEN** 用户请求"搜索/查找 XXX 图片"
- **THEN** Agent 识别为图片搜索意图并分发到 search_images 工具

#### Scenario: 命中生成 icon 意图
- **WHEN** 用户请求"为某张图生成相关 icon"
- **THEN** 系统识别为生成 icon 意图并分发到图生 icon 工具（图生图大模型）
- **AND** 工具调用以事件形式可见，并携带其异步任务标识

#### Scenario: 命中平台适配意图
- **WHEN** 用户请求"切尺寸/适配尺寸/按某平台/某广告位出图"
- **THEN** 系统识别为平台适配意图并分发到平台适配工具（智能路由：比例一致走裁剪快路径，否则图生图重绘）
- **AND** 走 AI 路径时其工具完成事件携带异步任务标识，前端据此插入占位并订阅进度

#### Scenario: 工具完成事件携带任务标识
- **WHEN** 某工具调用成功并产生了一个异步长任务（生图/二次调整/生成 icon/平台适配 AI 重绘/生视频/搜索图片）
- **THEN** 该工具的完成事件在结果数据中携带该任务的 id 及任务类型（如 generate/video）
- **AND** 前端据此即时插入占位并订阅该任务的进度流

#### Scenario: 非长任务工具不携带任务标识
- **WHEN** 工具调用不产生异步长任务（如列举尺寸、平台适配命中比例一致快路径即时返回、纯裁剪即时返回）
- **THEN** 其完成事件不携带长任务 id
- **AND** 前端不会为其插入占位骨架

#### Scenario: 白名单外请求被拒绝
- **WHEN** 用户请求不在预设意图内
- **THEN** 系统不调用任何工具并礼貌说明能力范围

#### Scenario: 预分类命中时注入意图提示
- **WHEN** 服务端确定性预分类以足够置信度判定用户文本命中某白名单意图
- **THEN** 系统在本轮调用会话模型前，向上下文注入一条结构化「意图提示」，标明预判意图与建议优先考虑的工具
- **AND** 该提示仅作引导，会话模型据其真实理解仍可选择不同工具或不调用工具

#### Scenario: 预分类未命中不阻断
- **WHEN** 预分类无法以足够置信度归类用户文本
- **THEN** 系统不注入意图提示，按既有流程交由会话模型自行判断
- **AND** 正常对话与边界判断不受影响

#### Scenario: 命中宣发文案生成意图
- **WHEN** 用户请求"给这张图写几条投放文案 / 想几句广告语"
- **THEN** 系统识别为宣发文案生成意图并分发到 generate_copy 工具

#### Scenario: 命中文字叠加意图
- **WHEN** 用户请求"给图N 右下角加个『立即预约』按钮 / 打个 8 折角标"
- **THEN** 系统识别为文字叠加意图并分发到 overlay_text 工具

#### Scenario: 命中批量变体意图
- **WHEN** 用户请求"这张图多出 4 个不同风格的版本"
- **THEN** 系统识别为批量变体意图并分发到 generate_variants 工具

#### Scenario: 命中合规审查意图
- **WHEN** 用户请求"这套素材能不能投 / 帮我看下有没有敏感词"
- **THEN** 系统识别为合规审查意图并分发到 review_compliance 工具
