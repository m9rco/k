## ADDED Requirements
### Requirement: 选区局部特征精确描述
系统 SHALL 支持对某张资产图的一个**用户选定区域**产出结构化的局部特征描述，用于精确定义"要改的是哪个主体/图层"。系统 SHALL 提供 `POST /api/session/{id}/assets/{assetId}/describe-region`，入参为归一化 bounding box（`x,y,w,h` ∈ [0,1]，及可选选区轮廓 mask）。服务端 SHALL 按 bbox 从该资产原图裁出选区像素，并以**固定的服务端"局部特征描述"指令**调用既有视觉分析器（默认 `gemini-2.5-flash-all`，inline base64，复用 `NeedsPublicURL()=false` 路径，COS 未配置亦可用）。

该指令 MUST 为服务端固定文案、**不嵌入任何用户自由文本**，要求模型**只描述选区内确有的要素、不虚构、不描述框外**，并产出结构化字段：主体类别、外观/材质/颜色、可见文字、在整图中的相对位置、必须保留的要素。视觉模型不可用（超时/未配置/网络错误）时，接口 SHALL 返回明确的降级信号，由调用方退回纯文本二次调整。

#### Scenario: 选区裁片产出结构化特征
- **WHEN** 客户端以归一化 bbox 调用 `describe-region`
- **THEN** 服务端按 bbox 裁出该资产选区像素并以局部描述指令调用视觉模型
- **AND** 返回含主体类别/外观材质颜色/可见文字/相对位置/必须保留的结构化特征文本

#### Scenario: 描述指令防注入且不虚构
- **WHEN** 组装局部特征描述请求
- **THEN** 描述指令完全由服务端固定文案构成，不嵌入用户自由文本
- **AND** 指令要求只描述选区内确有要素、不描述框外、不虚构

#### Scenario: 视觉模型不可用降级
- **WHEN** 视觉模型调用失败（超时/未配置/网络错误）
- **THEN** 接口返回明确的"描述不可用"信号
- **AND** 调用方回退到不含选区特征的纯文本二次调整

#### Scenario: 非法选区入参
- **WHEN** bbox 越界或非法（非 [0,1] 区间、宽高为 0）
- **THEN** 接口返回明确的入参错误，不调用视觉模型
